# Filosofi Brand & Palet Warna

Aplikasi **Tres Bros Caffè** mengusung identitas visual yang *earthy*, klasik, dan bernuansa kopi yang kental. Palet warna didesain agar terlihat profesional, menenangkan, sekaligus memberikan kesan premium bagi para pengguna aplikasi (kasir, barista, admin).

## Palet Warna Utama (Brand Colors)

| Nama Warna | Kode Hex | Penggunaan Utama |
| :--- | :---: | :--- |
| **Olive Green** | `#4B5A3A` | Warna *Primary*. Sangat cocok untuk *header*, tombol aksi utama (CTA), dan penanda status keberhasilan ("Selesai"). |
| **Sage Green** | `#7D8F6A` | Warna *Secondary*. Digunakan untuk elemen pendukung, *hover state*, latar belakang sekunder, atau indikator status "Antrean". |
| **Cream** | `#F3EDE1` | Warna *Surface / Background*. Jika menggunakan tema terang (*Light Mode*), warna ini menjadi kanvas utama. Pada *Dark Mode*, warna ini dipakai untuk teks primer agar visibilitasnya elegan. |
| **Warm Brown** | `#A16B3D` | Warna *Accent*. Warna yang hangat dan mencolok untuk menonjolkan fitur khusus, label harga, atau status "Sedang Dibuat (In Progress)". |
| **Dark Brown** | `#3A2B1F` | Warna *Dark Background*. Warna super gelap sebagai pengganti warna hitam murni. Digunakan sebagai kanvas utama di layar Kasir dan Dapur agar mata staf tidak cepat lelah (anti-glare). |

## Palet Warna Semantik (Untuk Sistem KDS & POS)

Untuk menjaga fungsionalitas dan ergonomi visual saat operasional:
*   **Antrean (To-Do)**: Menggunakan `Sage Green` (`#7D8F6A`). Memberi kesan "ada pesanan masuk, tapi tenang".
*   **Proses (In-Progress)**: Menggunakan `Warm Brown` (`#A16B3D`). Menandakan pesanan sedang "panas" (diseduh).
*   **Selesai (Done)**: Menggunakan `Olive Green` (`#4B5A3A`). Tanda aman bahwa orderan selesai dikerjakan.
*   **Urgensi (Alert/Delay)**: Menggunakan merah standar UI (`#C53030`) hanya saat pesanan melebihi batas waktu (SLA).
